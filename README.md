# Shader Hand Particle System

🎨 Interactive 3D particle system with hand gesture detection using Three.js and MediaPipe

## Features
- 🌌 8000 real-time particles with custom WebGL shaders
- ✨ Hand tracking with MediaPipe
- 🖐️ 4 gesture recognition modes:
  - **Point** - Attracts particles
  - **Pinch** - Strong attraction
  - **Open** - Repels particles
  - **Fist** - Weak repulsion
- 💫 Bloom post-processing effects
- 🌊 Dynamic wave and pulse effects

## How to Use
Open `index.html` in your browser. Allow camera access to enable hand detection.
